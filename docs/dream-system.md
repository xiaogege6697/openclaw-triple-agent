# 做梦系统设计文档

## 灵感来源：人类睡眠

本系统借鉴了人类大脑在睡眠期间的神经科学机制：

| 生物机制 | 系统对应 | 阶段 |
|----------|---------|------|
| **N2 睡眠纺锤波** | 记忆整合 | Phase 2 |
| **N3 慢波睡眠 (SWS)** | 短期→长期记忆转移 | Phase 2a/2b |
| **海马体 sharp-wave ripple** | ⭐重要标记优先回放 | Phase 3 |
| **突触等比例缩放** | 分层瘦身（合并→压缩→删除） | Phase 2c |
| **类淋巴清除** | 清理临时文件 | Phase 6 |
| **REM 预测编码** | 风险预警、趋势预测 | Phase 7 |
| **情绪调节（杏仁核）** | secretary 专用 | secretary Phase 2c |

## 设计原则

1. **只诊断不手术**：不自动修代码 bug，只报告
2. **增量记忆**：只追加不覆盖，归档不删除
3. **60行瘦身线**：MEMORY.md 控制在 60 行以内
4. **去重不丢信息**：分层瘦身策略

## 添加 ⭐ 标记

在对话中说"记住这个""很重要""别忘了"等关键词，
dream 系统会在 Phase 3 优先处理这些内容。
